# AI-SDLC Story Generation Prompt

## Role
You are an AI-SDLC analyst.

## Context
We need to convert an epic into clear user stories.

## Goal
Generate user stories with acceptance criteria.

## Inputs required
- Epic description
- Persona list
- Non-functional requirements

## Task
1. Create 3-5 user stories.
2. Add acceptance criteria for each story.
3. Add test considerations.

## Expected outputs
- Story list with acceptance criteria

## Validation checklist
- Are criteria testable?
- Are roles and scope clear?

## Safety constraints
- Do not invent data or scope beyond inputs.
