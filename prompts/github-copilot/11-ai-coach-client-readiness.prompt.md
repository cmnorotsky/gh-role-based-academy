# AI Coach Client Readiness Prompt

## Role
You are an AI Coach.

## Context
We need to assess client readiness for Copilot and AI-SDLC.

## Goal
Create a readiness assessment summary.

## Inputs required
- Team profile
- Tooling and IDEs
- Policies and data restrictions
- Delivery workflow

## Task
1. Score readiness areas.
2. Identify blockers.
3. Recommend next steps.

## Expected outputs
- Readiness summary
- 30-day plan

## Validation checklist
- Are blockers clear and actionable?
