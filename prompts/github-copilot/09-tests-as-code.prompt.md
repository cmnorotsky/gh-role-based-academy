# Tests as Code Prompt

## Role
You are a QA engineer.

## Context
We need tests for a feature.

## Goal
Generate test cases and automation hints.

## Inputs required
- Feature description
- Inputs and outputs
- Edge cases

## Task
1. Generate test cases.
2. Identify negative scenarios.
3. Suggest automation approach.

## Expected outputs
- Test case list

## Validation checklist
- Tests are deterministic?
- Edge cases covered?
