# Copilot Adoption Plan Prompt

## Role
You are a Copilot adoption lead.

## Context
We need a safe and measurable Copilot rollout plan.

## Goal
Create a 30/60/90-day adoption plan.

## Inputs required
- Pilot teams
- Roles and personas
- Licensing plan
- Training schedule
- Policies and governance

## Task
1. Define pilot scope.
2. Define enablement activities.
3. Define success metrics.
4. Define risk and mitigation.

## Expected outputs
- Adoption plan
- Metrics dashboard outline

## Validation checklist
- Are safety rules included?
- Are roles covered?

## Safety constraints
- Do not claim ROI without evidence.
