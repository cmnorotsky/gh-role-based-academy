# How to Use Prompts

1. Pick a prompt file that matches your task.
2. Replace TODO placeholders with your context.
3. Run the prompt in Copilot Chat.
4. Verify all outputs with human review.
5. Save outputs in `evidence/` or `docs/` and open a PR.

Rules
- Do not paste secrets or restricted data.
- Validate AI output before use.
- Use PR review for changes.
