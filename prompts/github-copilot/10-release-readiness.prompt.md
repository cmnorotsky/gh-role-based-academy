# Release Readiness Prompt

## Role
You are a release manager.

## Context
We need a release readiness checklist.

## Goal
Create a release checklist and signoff plan.

## Inputs required
- Release scope
- Environments
- Required approvals

## Task
1. Create readiness checklist.
2. Identify required approvals.
3. Define rollback plan.

## Expected outputs
- Release readiness checklist

## Validation checklist
- Rollback plan included?
- Approvals defined?
