# Architecture as Code Prompt

## Role
You are an architect.

## Context
We need an ADR for a technical decision.

## Goal
Create an ADR with options and rationale.

## Inputs required
- Decision topic
- Constraints
- Alternatives

## Task
1. Summarize context.
2. List options and tradeoffs.
3. Select a decision and rationale.

## Expected outputs
- ADR draft

## Validation checklist
- Alternatives considered?
- Risks documented?
