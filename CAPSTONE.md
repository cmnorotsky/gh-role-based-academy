# Capstone

## Scenario
A fictional enterprise has:
- 500 repos
- 60 Azure DevOps projects
- 2,000 pipelines
- 100 classic release pipelines
- Mixed .NET, Java, Node, Python, Terraform
- 20 high-risk repos
- Existing secrets in history
- Manual release approvals
- Azure deployments
- Product teams using Azure Boards
- Leadership wants GitHub Enterprise Cloud, GHAS, Copilot, and AI-SDLC

## Learner deliverables
- Discovery report
- Migration wave plan
- Governance model
- GHAS rollout plan
- Copilot adoption plan
- AI-SDLC model
- Pilot implementation plan
- Executive readout
- Go/no-go recommendation

## AI Coach Capstone Add-On

In addition to the migration and GitHub implementation deliverables, the learner must create an AI Coach enablement plan for the fictional enterprise.

The plan must include:

1. Client AI readiness assessment
2. Pilot team selection
3. Developer workshop plan
4. BA/Product workshop plan
5. QA workshop plan
6. DevOps workshop plan
7. Security-aware AI usage guide
8. Office hours model
9. Prompt library
10. Copilot usage and value report
11. Client champion handoff plan
12. 30/60/90-day enablement roadmap

The learner must present how AI coaching will help the client move from Copilot licensing to measurable engineering behavior change.
