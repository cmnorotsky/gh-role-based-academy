# Certification Path

This repo aligns to GitHub certifications. Use the labs below as readiness evidence.

## GitHub Foundations
- Target audience: All roles
- Prerequisites: Basic command line and Git concepts
- Skills to learn: Repos, issues, PRs, branches, Projects
- Supporting labs: docs/01-foundations/lab-01-git-basics.md, lab-02-branching-prs-reviews.md, lab-03-issues-projects-milestones.md
- Readiness checklist: Complete labs; open and review PR; use Projects and milestones
- Practice deliverable: PR with a completed learning evidence file

## GitHub Actions
- Target audience: App devs, DevOps, platform engineers
- Prerequisites: Foundations certification or equivalent
- Skills to learn: Workflows, reusable workflows, environments, OIDC
- Supporting labs: docs/03-actions/lab-01-basic-ci.md through lab-06-migrate-ado-pipeline.md
- Readiness checklist: Build workflow, reusable workflow, environment approvals
- Practice deliverable: Workflow PR and run log screenshot

## GitHub Administration
- Target audience: Platform, DevOps, GitHub admins
- Prerequisites: Foundations
- Skills to learn: Org model, teams, EMU, rulesets, audit log
- Supporting labs: docs/02-administration/lab-01-enterprise-org-repo-model.md through lab-04-audit-log-and-governance.md
- Readiness checklist: Proposed org model, ruleset plan, audit log review
- Practice deliverable: Governance model draft

## GitHub Advanced Security
- Target audience: Security, app teams, DevSecOps
- Prerequisites: Foundations, Actions basics
- Skills to learn: CodeQL, secret scanning, dependency review, triage
- Supporting labs: docs/05-ghas/lab-01-enable-security-features.md through lab-07-security-metrics.md
- Readiness checklist: CodeQL workflow run, triage notes, remediation plan
- Practice deliverable: Security enablement plan

## GitHub Copilot
- Target audience: All roles, enablement leads
- Prerequisites: Foundations
- Skills to learn: Copilot features, prompt patterns, safe usage, metrics
- Supporting labs: docs/06-copilot/lab-01-copilot-setup.md through lab-07-adoption-playbook.md
- Readiness checklist: Copilot setup verified, prompt library created, usage model defined
- Practice deliverable: Copilot adoption plan and value report sample
