# Lab 01: Migration Discovery

## Objective
Gather discovery inputs for migration planning.

## Target audience
Migration factory, DevOps

## Prerequisites
- Actions module overview

## Estimated time
2 hours

## Tools required
- Inventory templates

## Step-by-step lab
1. Identify ADO orgs and projects.
2. List repos, pipelines, and release pipelines.
3. Identify business critical apps.
4. Capture ownership and contacts.
5. Document constraints and deadlines.

## Common mistakes
- Missing ownership or criticality

## Evidence to submit
- Discovery summary document

## Review checklist
- Critical systems identified

## Definition of done
- Discovery summary approved
