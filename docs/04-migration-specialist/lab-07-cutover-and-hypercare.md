# Lab 07: Cutover and Hypercare

## Objective
Plan cutover, validation, and hypercare.

## Target audience
Migration factory

## Prerequisites
- Labs 01-06

## Estimated time
2 hours

## Tools required
- Migration templates

## Step-by-step lab
1. Define cutover window and freeze.
2. Create validation checklist.
3. Define hypercare support model.
4. Define rollback triggers and steps.

## Common mistakes
- No rollback plan

## Evidence to submit
- Cutover and hypercare plan

## Review checklist
- Validation and rollback included

## Definition of done
- Plan approved by review board
