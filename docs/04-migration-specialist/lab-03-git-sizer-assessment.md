# Lab 03: Git Sizer Assessment

## Objective
Assess large repos and history risks.

## Target audience
Migration factory

## Prerequisites
- Lab 02

## Estimated time
2 hours

## Tools required
- git-sizer (or wrapper script)

## Step-by-step lab
1. Run git-sizer on sample repos.
2. Record size and object counts.
3. Identify history cleanup needs.
4. Flag high-risk repos.

## Common mistakes
- Ignoring large files in history

## Evidence to submit
- Git sizer results table

## Review checklist
- Risks captured with mitigation

## Definition of done
- High-risk repos identified
