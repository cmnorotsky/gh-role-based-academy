# Migration Specialist Module Overview

## Objective
Prepare for Azure DevOps to GitHub migration planning.

## Target audience
DevOps, platform, migration factory

## Prerequisites
- Foundations, Actions basics

## Estimated time
10-12 hours

## Labs
1. Migration discovery
2. Repo inventory
3. Git sizer assessment
4. GEI planning
5. Pipeline migration plan
6. Boards/work items strategy
7. Cutover and hypercare

## Deliverables
- Discovery report and wave plan

## Review criteria
- Plans are realistic and include validation and rollback
