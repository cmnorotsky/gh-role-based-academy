# Lab 06: Boards and Work Items Strategy

## Objective
Plan work item strategy during migration.

## Target audience
Migration factory, BA/Product

## Prerequisites
- Lab 01

## Estimated time
2 hours

## Tools required
- ADO Boards and GitHub Issues

## Step-by-step lab
1. Decide dual-tracking or cutover approach.
2. Map ADO work items to GitHub Issues.
3. Define linking strategy for PRs.
4. Document transition timeline.

## Common mistakes
- No clear ownership of work item changes

## Evidence to submit
- Boards transition plan

## Review checklist
- Stakeholders aligned

## Definition of done
- Transition plan approved
