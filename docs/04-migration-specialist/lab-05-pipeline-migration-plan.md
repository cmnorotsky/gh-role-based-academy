# Lab 05: Pipeline Migration Plan

## Objective
Plan ADO pipeline migration to GitHub Actions.

## Target audience
Migration factory, DevOps

## Prerequisites
- Actions module

## Estimated time
2-3 hours

## Tools required
- Actions Importer (if available)

## Step-by-step lab
1. Inventory ADO pipelines.
2. Run an importer analysis or simulate.
3. Identify manual conversion tasks.
4. Define pipeline conversion waves.

## Common mistakes
- Overlooking environment approvals

## Evidence to submit
- Pipeline migration plan

## Review checklist
- Manual tasks and gaps documented

## Definition of done
- Plan reviewed by DevOps lead
