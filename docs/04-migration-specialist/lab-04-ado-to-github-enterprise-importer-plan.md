# Lab 04: GEI Planning

## Objective
Plan repository migration using GitHub Enterprise Importer.

## Target audience
Migration factory

## Prerequisites
- Lab 01

## Estimated time
2 hours

## Tools required
- GEI docs, GitHub CLI

## Step-by-step lab
1. Review GEI capabilities and limits.
2. Identify repos suitable for automated migration.
3. Document manual steps required.
4. Draft a GEI runbook.

## Common mistakes
- Assuming issues and PRs migrate without validation

## Evidence to submit
- GEI plan and runbook

## Review checklist
- Manual steps called out

## Definition of done
- Runbook approved
