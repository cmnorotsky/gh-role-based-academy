# How to Install Tools

## Objective
Install the required tools for GitHub training.

## Tools required
- Git
- GitHub CLI
- VS Code
- GitHub Copilot extension
- GitHub Pull Requests extension

## Steps
1. Install Git from https://git-scm.com.
2. Install GitHub CLI from https://cli.github.com.
3. Install VS Code from https://code.visualstudio.com.
4. Install the extensions in VS Code.
5. Run `gh auth login` and verify with `gh auth status`.

## Evidence to submit
- Screenshot or note of `gh auth status`

## Definition of done
- Tools are installed and GitHub CLI is authenticated.
