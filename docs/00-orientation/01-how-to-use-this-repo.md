# How to Use This Repo

## Objective
Learn how the academy is organized and how to submit evidence.

## Steps
1. Read [README.md](../../README.md) and [ROLE-MATRIX.md](../../ROLE-MATRIX.md).
2. Identify your role and required modules.
3. Open the project board and find the M00 Orientation issues.
4. Create a learner branch: `learner/<your-name>/orientation`.
5. Create an evidence file: `evidence/orientation/<your-name>.md`.
6. Submit a PR for review.

## Evidence to submit
- PR link with your evidence file
- Short reflection on learning goals

## Definition of done
- Mentor approves the PR and confirms your learning path.
