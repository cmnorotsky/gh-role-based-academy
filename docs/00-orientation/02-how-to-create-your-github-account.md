# How to Create Your GitHub Account

## Objective
Ensure you have the correct GitHub identity for the enterprise.

## Steps
1. Determine whether your org uses personal accounts or EMU.
2. If EMU, follow enterprise instructions for account creation.
3. Verify access to the training organization.
4. Enable MFA if required.

## Evidence to submit
- Confirmation that you can access the training org

## Definition of done
- You can log in to GitHub and view the training org.
