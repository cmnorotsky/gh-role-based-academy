# How to Work Through Labs

## Objective
Understand lab expectations and evidence submission.

## Steps
1. Read the lab instructions fully before starting.
2. Create a branch for each lab or module.
3. Capture evidence as markdown in `evidence/<module>/<your-name>.md`.
4. Open a PR and request mentor review.
5. Address review comments and merge.

## Evidence to submit
- PR link with evidence
- Notes on blockers or questions

## Definition of done
- Lab evidence is approved by a mentor.
