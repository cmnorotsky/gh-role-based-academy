# Lab 02: Architecture as Code

## Objective
Create an ADR and architecture diagram.

## Target audience
Architects, leads

## Prerequisites
- Lab 01

## Estimated time
2-3 hours

## Tools required
- Markdown, diagram tool

## Step-by-step lab
1. Create an ADR under `docs/architecture/`.
2. Add context, decision, and consequences.
3. Add a diagram link or placeholder.
4. Open a PR for review.

## Common mistakes
- No alternatives considered

## Evidence to submit
- ADR PR link

## Review checklist
- Decision is clear and reviewed

## Definition of done
- ADR approved
