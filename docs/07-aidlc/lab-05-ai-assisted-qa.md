# Lab 05: AI-Assisted QA

## Objective
Use AI to propose test cases and edge cases.

## Target audience
QA, developers

## Prerequisites
- Lab 03

## Estimated time
2 hours

## Tools required
- Copilot Chat

## Step-by-step lab
1. Select a feature.
2. Ask for test cases and negative scenarios.
3. Validate tests with team.
4. Add approved tests.

## Common mistakes
- Treating AI output as final

## Evidence to submit
- Test case list and approvals

## Review checklist
- Tests validated by humans

## Definition of done
- Tests added or documented
