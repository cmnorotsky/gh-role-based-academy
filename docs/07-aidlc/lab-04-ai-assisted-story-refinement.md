# Lab 04: AI-Assisted Story Refinement

## Objective
Use AI to refine stories and acceptance criteria.

## Target audience
BA/Product, QA

## Prerequisites
- Lab 01

## Estimated time
2 hours

## Tools required
- Copilot Chat

## Step-by-step lab
1. Select a vague story.
2. Ask Copilot for clarifying questions.
3. Update acceptance criteria.
4. Add test considerations.

## Common mistakes
- Accepting AI output without review

## Evidence to submit
- Updated story and notes

## Review checklist
- Acceptance criteria are testable

## Definition of done
- Story refinement approved
