# Lab 03: Tests as Code

## Objective
Treat tests as first-class artifacts.

## Target audience
Developers, QA

## Prerequisites
- Lab 01

## Estimated time
2 hours

## Tools required
- Test framework

## Step-by-step lab
1. Identify a feature and its tests.
2. Add or update tests in a PR.
3. Document test coverage improvements.

## Common mistakes
- No evidence of test execution

## Evidence to submit
- PR link and test run output

## Review checklist
- Tests run in CI

## Definition of done
- Tests merged with review
