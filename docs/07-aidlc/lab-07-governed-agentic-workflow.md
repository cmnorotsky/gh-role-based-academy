# Lab 07: Governed Agentic Workflow

## Objective
Define guardrails for agentic workflows.

## Target audience
Architects, platform, security

## Prerequisites
- AI-SDLC module overview

## Estimated time
2 hours

## Tools required
- Governance templates

## Step-by-step lab
1. Define tasks eligible for agents.
2. Define human approval checkpoints.
3. Define required scans and tests.
4. Document escalation process.

## Common mistakes
- Allowing agent output to bypass reviews

## Evidence to submit
- Agent governance policy

## Review checklist
- Human gates are enforced

## Definition of done
- Policy approved by governance board
