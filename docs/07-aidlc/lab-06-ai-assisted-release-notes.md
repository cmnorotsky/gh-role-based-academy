# Lab 06: AI-Assisted Release Notes

## Objective
Use AI to draft release notes from PRs.

## Target audience
Developers, release managers

## Prerequisites
- Foundations

## Estimated time
1-2 hours

## Tools required
- Copilot Chat, GitHub PRs

## Step-by-step lab
1. Select recent merged PRs.
2. Ask Copilot to draft release notes.
3. Verify accuracy and edit.

## Common mistakes
- Publishing AI notes without validation

## Evidence to submit
- Draft release notes and edits

## Review checklist
- Accuracy verified by human

## Definition of done
- Release notes approved
