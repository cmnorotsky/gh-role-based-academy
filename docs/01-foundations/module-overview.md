# Foundations Module Overview

## Objective
Build GitHub and Git fundamentals for day-to-day delivery.

## Target audience
All roles

## Prerequisites
- Orientation complete

## Estimated time
6-8 hours

## Labs
1. Git basics
2. Branching, PRs, and reviews
3. Issues, Projects, milestones
4. Repo settings and protections

## Deliverables
- Evidence PRs for each lab

## Review criteria
- Learner demonstrates core GitHub workflows
