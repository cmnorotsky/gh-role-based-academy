# Lab 01: Git Basics

## Objective
Learn clone, branch, commit, and push.

## Target audience
All roles

## Prerequisites
- Tools installed

## Estimated time
2 hours

## Tools required
- Git, GitHub CLI, VS Code

## Step-by-step lab
1. Clone the repo.
2. Create a branch named `learner/<your-name>/git-basics`.
3. Create a file: `evidence/foundations/git-basics-<your-name>.md`.
4. Add a short note about Git commands used.
5. Commit and push.
6. Open a PR.

## Common mistakes
- Committing directly to main
- Forgetting to push the branch

## Evidence to submit
- PR link
- Evidence file with commands used

## Review checklist
- Branch naming is correct
- Evidence file exists

## Definition of done
- PR approved by mentor
