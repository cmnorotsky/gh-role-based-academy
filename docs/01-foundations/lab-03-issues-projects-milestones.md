# Lab 03: Issues, Projects, Milestones

## Objective
Use issues and Projects for planning.

## Target audience
All roles

## Prerequisites
- Lab 01

## Estimated time
1-2 hours

## Tools required
- GitHub web UI

## Step-by-step lab
1. Create an issue using the learning story template.
2. Add labels for track and role.
3. Assign the issue to the M01 Foundations milestone.
4. Add the issue to the project board.
5. Move the issue across status columns.

## Common mistakes
- Missing labels or milestone

## Evidence to submit
- Issue link
- Screenshot of project board with your issue

## Review checklist
- Issue uses the template
- Labels and milestone are correct

## Definition of done
- Issue is tracked on the project board
