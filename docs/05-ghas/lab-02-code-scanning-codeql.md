# Lab 02: Code Scanning with CodeQL

## Objective
Run CodeQL analysis on a sample repo.

## Target audience
Security, app teams

## Prerequisites
- Lab 01

## Estimated time
2-3 hours

## Tools required
- GitHub Actions

## Step-by-step lab
1. Add a CodeQL workflow from sample.
2. Select target languages.
3. Run the workflow on a PR.
4. Review results and triage one alert.

## Common mistakes
- Skipping baseline triage notes

## Evidence to submit
- PR link and triage notes

## Review checklist
- Alert ownership assigned

## Definition of done
- CodeQL workflow runs successfully
