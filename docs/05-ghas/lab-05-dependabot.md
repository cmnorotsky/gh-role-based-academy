# Lab 05: Dependabot

## Objective
Configure Dependabot alerts and updates.

## Target audience
Security, app teams

## Prerequisites
- Lab 01

## Estimated time
1-2 hours

## Tools required
- GitHub web UI

## Step-by-step lab
1. Add a Dependabot config file.
2. Define update schedules.
3. Review alert settings.

## Common mistakes
- Overly frequent update noise

## Evidence to submit
- Dependabot config file

## Review checklist
- Update cadence is reasonable

## Definition of done
- Dependabot enabled
