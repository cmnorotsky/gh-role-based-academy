# Lab 06: Security Campaign

## Objective
Plan a security remediation campaign.

## Target audience
Security leads

## Prerequisites
- Lab 02

## Estimated time
2 hours

## Tools required
- Security overview

## Step-by-step lab
1. Select a target set of alerts.
2. Define scope, owners, and timelines.
3. Define tracking and reporting.

## Common mistakes
- No owner for remediation

## Evidence to submit
- Campaign plan

## Review checklist
- Scope and owners defined

## Definition of done
- Campaign plan approved
