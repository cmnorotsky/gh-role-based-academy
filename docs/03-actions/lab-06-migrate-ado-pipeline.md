# Lab 06: Migrate ADO Pipeline

## Objective
Assess and plan a pipeline migration.

## Target audience
DevOps, migration team

## Prerequisites
- Labs 01-03

## Estimated time
2-3 hours

## Tools required
- GitHub Actions Importer (if available)

## Step-by-step lab
1. Inventory a sample ADO pipeline.
2. Identify tasks that map to Actions.
3. Identify manual conversion steps.
4. Draft a GitHub Actions workflow.
5. Document gaps and risks.

## Common mistakes
- Assuming full automation for complex release gates

## Evidence to submit
- Draft workflow and gap list

## Review checklist
- Manual steps clearly called out

## Definition of done
- Migration plan approved by mentor
