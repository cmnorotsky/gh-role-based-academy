# Actions Module Overview

## Objective
Build CI/CD using GitHub Actions and reusable workflows.

## Target audience
App devs, DevOps, platform engineers

## Prerequisites
- Foundations module

## Estimated time
8-10 hours

## Labs
1. Basic CI
2. Reusable workflows
3. Environments and approvals
4. OIDC to Azure
5. Self-hosted runners
6. Migrate ADO pipeline

## Deliverables
- Workflow PRs and run logs

## Review criteria
- Workflows are secure and maintainable
