# Lab 01: Basic CI

## Objective
Create a basic CI workflow.

## Target audience
App devs, DevOps

## Prerequisites
- Foundations module

## Estimated time
2 hours

## Tools required
- GitHub Actions

## Step-by-step lab
1. Create `.github/workflows/basic-ci.yml` in a branch.
2. Add steps for checkout, build, and test.
3. Open a PR.
4. Verify workflow run.

## Common mistakes
- Missing `actions/checkout`

## Evidence to submit
- PR link and workflow run URL

## Review checklist
- Workflow runs successfully

## Definition of done
- CI workflow passes
