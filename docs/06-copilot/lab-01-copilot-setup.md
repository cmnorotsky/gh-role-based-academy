# Lab 01: Copilot Setup

## Objective
Verify Copilot access and IDE setup.

## Target audience
All roles

## Prerequisites
- Orientation complete

## Estimated time
1-2 hours

## Tools required
- VS Code, Copilot extension

## Step-by-step lab
1. Confirm license assignment.
2. Sign in to GitHub in VS Code.
3. Open Copilot Chat.
4. Test inline suggestions.

## Common mistakes
- Using an unlicensed account

## Evidence to submit
- Screenshot or note confirming Copilot works

## Review checklist
- Copilot is available in IDE

## Definition of done
- Copilot is active for the learner
