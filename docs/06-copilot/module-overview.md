# Copilot Module Overview

## Objective
Enable safe Copilot adoption and value measurement.

## Target audience
All roles, enablement leads

## Prerequisites
- Foundations module

## Estimated time
8-10 hours

## Labs
1. Copilot setup
2. Prompt patterns
3. Copilot for tests
4. Copilot for code review
5. Copilot for infra
6. Measure usage and value
7. Adoption playbook

## Deliverables
- Adoption plan and metrics model

## Review criteria
- Safe usage guidance and measurable outcomes
