# Lab 02: Prompt Patterns

## Objective
Learn prompt patterns for common tasks.

## Target audience
All roles

## Prerequisites
- Lab 01

## Estimated time
2 hours

## Tools required
- Copilot Chat

## Step-by-step lab
1. Create a prompt for code explanation.
2. Create a prompt for refactoring.
3. Create a prompt for test ideas.
4. Create a prompt for documentation.
5. Document good and bad prompt examples.

## Common mistakes
- Vague prompts without context

## Evidence to submit
- Prompt library markdown

## Review checklist
- Prompts are specific and safe

## Definition of done
- Prompt library reviewed by mentor
