# Lab 03: Copilot for Tests

## Objective
Use Copilot to assist with test creation.

## Target audience
Developers, QA

## Prerequisites
- Lab 01

## Estimated time
2 hours

## Tools required
- Copilot Chat, test framework

## Step-by-step lab
1. Select a simple function.
2. Ask Copilot for test cases.
3. Implement tests and run them.
4. Record any adjustments.

## Common mistakes
- Accepting tests without running them

## Evidence to submit
- Test file and run results

## Review checklist
- Tests are executed and pass

## Definition of done
- Tests added with evidence
