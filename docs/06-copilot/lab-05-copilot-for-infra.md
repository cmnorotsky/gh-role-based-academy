# Lab 05: Copilot for Infrastructure

## Objective
Use Copilot for IaC and scripting safely.

## Target audience
Infra and DevOps

## Prerequisites
- Lab 01

## Estimated time
2 hours

## Tools required
- Copilot Chat, IaC tooling

## Step-by-step lab
1. Select an IaC task.
2. Ask Copilot for a starter template.
3. Review for security and compliance.
4. Run in a sandbox only.

## Common mistakes
- Using AI-generated IaC in production without review

## Evidence to submit
- Template and review notes

## Review checklist
- Security review completed

## Definition of done
- Safe IaC draft produced
