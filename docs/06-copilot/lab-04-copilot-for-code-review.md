# Lab 04: Copilot for Code Review

## Objective
Use Copilot to draft review notes and PR summaries.

## Target audience
Developers, reviewers

## Prerequisites
- Lab 02

## Estimated time
1-2 hours

## Tools required
- Copilot Chat, GitHub PR

## Step-by-step lab
1. Open a PR.
2. Ask Copilot to summarize changes.
3. Draft review checklist items.
4. Validate findings with your own review.

## Common mistakes
- Trusting AI review without verification

## Evidence to submit
- PR summary and review notes

## Review checklist
- Human verification documented

## Definition of done
- Review notes are accurate and verified
