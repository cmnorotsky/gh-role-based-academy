# Lab 06: AI-Assisted Refactoring and Testing

## Objective
Use AI to refactor code with tests.

## Target audience
AI Coach, developers

## Prerequisites
- Lab 04

## Estimated time
3 hours

## Tools required
- Copilot Chat, test framework

## Step-by-step lab
1. Select a small refactor task.
2. Ask Copilot for a refactor plan.
3. Generate or update tests.
4. Run tests and document results.

## Evidence to submit
- Before/after summary and test results

## Definition of done
- Refactor validated by tests
